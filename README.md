# Forkify Recipe App

A recipe search app using external APIs to explore and save your favorite recipes.

## Features
- Search for recipes  
- View recipe details  
- Pagination of results  
- Bookmark favorite recipes  

## Tech Stack
- JavaScript (ES6+)  
- HTML & CSS  
- API Integration
